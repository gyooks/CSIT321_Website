# Epic Game Landing Page

🎮 A sleek, responsive gaming-themed landing page built with React, Vite, and Tailwind CSS.

## 🚀 Setup
```bash
npm install
npm run dev
```

## 📦 Build for Production
```bash
npm run build
```

## 📤 Deploy to GitHub Pages
Use a plugin like `vite-plugin-gh-pages`, or manually push the `dist/` folder to the `gh-pages` branch.
